INFORMATIO ABOUT DATASET:
The dataset encompasses athelete data of thier Career spanning from soccer 15 to soccer 21. It includes information on every player featured in soccer 21, covering over 100 attributes, positions, and player statistics such as Attacking, Skills, Defense, Mentality, GK Skills, and more.

feature in the app:
1. Contrasting the perfomances of atheletes from differnt generations
2. examing potentials of different atheletes and parity between them
3. Analyse performance of different franchises in different soccer tournaments
4. Examing the potential of different franchises based on age group of athelete they have


Prerequisites:
Make sure you have Python 3.6 or a newer version installed, as well as an internet browser.

Installation:
Before launching the application, ensure that you have the necessary libraries installed. You can do this by running the command:
pip install -r requirements.txt


Ensure you're in the project's root directory when executing the command, and make sure the requirements.txt file is in the same directory.

Running the Application:
Once the required libraries are installed, you can start the application by running:
python app.py

You can then access the application through your web browser at http://localhost:8050/ or via the address shown in the terminal.

Please note that the initial loading of the application might take some time as it fetches data from Google Drive. Kindly be patient during this process.
